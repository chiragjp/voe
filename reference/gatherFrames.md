# Gather and harmonize all vibration results

Gather and harmonize all vibration results

## Usage

``` r
gatherFrames(returnFrames)
```

## Arguments

- returnFrames:

  List of results from
  [`conductVibrationForK()`](https://chiragjp.github.io/voe/reference/conductVibrationForK.md).

## Value

A list with `vibFrame`, `bicFrame`, `combinations`, `adjust`, `family`,
and `base_formula`.

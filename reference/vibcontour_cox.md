# Contour volcano plot for Cox VoE

Contour volcano plot for Cox VoE

## Usage

``` r
vibcontour_cox(vibObj, factor_num = 1, alpha = 1)
```

## Arguments

- vibObj:

  A VoE result object.

- factor_num:

  Factor level to plot (default 1).

- alpha:

  Point transparency (default 1).

## Value

A
[`ggplot2::ggplot()`](https://ggplot2.tidyverse.org/reference/ggplot.html)
object.

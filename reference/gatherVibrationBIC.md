# Gather BIC results across k values

Gather BIC results across k values

## Usage

``` r
gatherVibrationBIC(returnFrames)
```

## Arguments

- returnFrames:

  List of results from
  [`conductVibrationForK()`](https://chiragjp.github.io/voe/reference/conductVibrationForK.md).

## Value

A matrix of BIC results with a `k` column appended.

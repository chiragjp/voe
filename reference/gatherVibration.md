# Gather vibration results across k values

Gather vibration results across k values

## Usage

``` r
gatherVibration(returnFrames)
```

## Arguments

- returnFrames:

  List of results from
  [`conductVibrationForK()`](https://chiragjp.github.io/voe/reference/conductVibrationForK.md).

## Value

A matrix of vibration results with a `k` column appended.
